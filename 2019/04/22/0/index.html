人格

人际交往

  • 对于长辈，如果是值得尊重的人，那便要努力的刷自己的存在感，要让他们知道自己的向上精神。要表现出自己不怕苦，不怕累的精神。这样才可能获得他们更多的帮助。对于以后的研究生老师更是如此。
    • 这是张宇老师在讲课中提到的，也是很多老师都提到过的问题。你的努力，老师都看的见，你的懒惰，老师也都看的见，只是不说而已。但他们会用实际行动告诉你，他们所知道的一切。
  • 学校是以团队还是个人成绩进行评估，决定了老师的态度。这是选择学校和老师的一道标准。
    • 鲁明羽老师在课上提到的。
  • 金芝老师从大学开始，就像是开了挂一样的，一路前行。她的履历证明，努力的人，上天永远不会亏待她。
    • 鲁明羽老师在课上提到了这位杰青，长江学者。
  • 我会不会在与他人的交流中倾向于解决他人的问题，而不是单纯的倾听。这也许不是他人述说，想要得到的结果。要警惕自己的这一行为。《男人来自火星，女人来自金星》中也提到了这一问题，看来这样的问题不止发生在异性的交流中，也发生在同性的交流中。
    • 我的玻璃碗碎了，我向舍友倾诉，今天是我的水逆日。但他们似乎只想要说服我买一个不会碎的碗，而不是倾听。况且是在我刚刚已经在网上买了一个同样碗的情况下。
  • 目前与父母的交往过程中，不能采用将心比心的方式。因为我还没有孩子，所以无法体谅做父母的心情，但是通过其他父母的表达，可以知道，我作为子女，要懂得在适当时间表达对于他们的关心和爱，虽然他们嘴上总是说不用管他们。
    • 来自《傅雷家书》中朱梅馥对傅聪写的一封信。

品行修养

  • 如果意识到自己的错误，就应该积极的弥补和改正，以保证下一次可以在犯之后更快的意识到，以保证自己可以在下下次犯之中意识到，以保证自己可以在下下下次没有范之前意识到。以保证自己以后都不会再范。

    • 也就是说，我面对同一个问题，只有三次机会可以被宽容。超过三次，我就该思考，我是不是对于这个问题太过轻视，或者是对于自己要求不够严苛。

  • 谨言慎行：动嘴永远比动脑快，这是我的大毛病。为了这个事情，不知道已经得罪了多少人。就是没有办法戒掉这个坏毛病，以后不要再多说话了，慢一点思考，慢一步说话总是好的。

  • 不善良：总有恶魔在我心中，我不愿意善良，也不愿意是个坏人，所以最好是不做坏事，也不做好事。

  • 自视过高：我总是自视过高，认为周围的人比不上自己，这也是大毛病，只要这样的想法出现，我就不能正常对待同龄人，我会下意识的去做一些伤害别人的事情甚至我自己都还没有意识到。这个是潜意识，是观念上的错误，如果不改的话，会危害我一辈子。

    到底要怎么改，我还没有想到办法，因为到现在为止，我还是认为我周围的人没有我的思维深度深，但是我根本就不知道他们的思考深度，我又如何能够断言他们的想法呢？如果是生活很久的朋友，我可以直接下结论，因为我已经观察他们很久了。但是不认识的人，希望自己一律保持尊敬的态度，不要再生活中应用类比假设，没有交往的人，请保持尊重。当然这也不代表着，如果是思想不和我在一个水平线上的人，我就可以不尊重，只是我们不在一个层次上，要么疏远，要么就学别人好的地方，不要自视过高。

  • 急躁，这是在没有意识到大时间周期前的时候自己的一个大毛病，但是现在自己一直在积极的改正当中。

    • 也许可以养一株植物来治一治我的急躁。

  • 学会情绪管理和表情管理。

    • 不要总是轻易的外露自己的情绪，也不要将自己的喜欢厌恶表现在脸上，这个方面我完全没有想到解决办法，只能以后找到办法以后再说。
    • 人总是易于迁怒，因为对自己的不满意，而将情绪转移到别人身上，这样的事情一定要警惕，这是很伤害自己的事情。如果真的是纯粹的讨厌一个人，那就远离那个人，不要接近，甚至是在别人接近的时候做好表情管理。
    • 我们总是会因为赌气而在无意识间放弃很多东西 和母亲通话 谈到自习室的讨厌的人的问题 母亲说要我搬 我说我先来的为什么搬 为什么不是别人搬 然后难过的还是自己 这个问题要自己审视一下
    • 非理智的情绪总是会导致我视野的狭隘，考虑什么是非理智情绪，意识到自己的非理智情绪，观察这种情绪对于自己造成的伤害，克服或者说是去感受，然后放松，等着情绪的自我消散。

  • 默棋法：对于未发生事件的预判能力和处理机制应该建立，而不是什么都不知道的面对任何未知事件。在真正面对一件事之前，应该在脑海中演算很多次，来考虑到每一种可能性和可能的结果。

    • 类似于王阳明和曾国藩的军事行动。类似于围棋之道。这些都需要我更多的阅历和经历。
    • 默棋中要记入恋爱，考虑恋爱以后各种情况发生的可能性，以及相应的处理方式，以及可能预料的结果，这些都需要我更为多的阅历和经历
    • 默棋中记入困难时刻，比如困倦时刻，比如出现快递以后自己不想待在教室。比如和同学一起吃饭以后想回寝室睡觉。比如天气越来越冷以后，没有勇气去外面学习，比如题目很难，没有办法学习。自己没有再规定时间完成计划。他人要求我去完成别的事情
    • 英语水平得不到提高。
    • 我的独立性还能够保持多久，五年，十年，二十年，还是一辈子。如果没有保持，会是什么原因导致的，又会是什么原因造就我的一直持续。
    • 我会在什么时候放弃写作，放弃写文章，为什么我现在觉得极其正确的事情，会在时间的腐朽下褪色。
    • 最大的麻烦是，当大家都在休息和玩的时候，自己是否还可以坚守自己的理想和努力状态，如果不能坚守的话，要怎么办才可以坚守

  • 反向思维方式：当某一样东西对于自己来说是很好的东西，我不应该盲目的赞扬，而应该更加理智的去考虑这个东西的弊端在哪里。

  • 面对诱惑的处理方式：面对诱惑的习惯性处理思维应该形成，而不是没有任何防御机制的面对。做的不好。

  • 嫉妒：我的身边我的周围没有看到有趣有努力的灵魂，一是由于我与其他人交流太少二是我身处的环境不够优秀 三是我本身就害怕出现优秀的人，因为那些人会引起我强烈的嫉妒，而我到目前为止还没有解决嫉妒的办法。

    • 攀比心理，好强是好事，但是我总是会让这种情绪对我造成伤害，这就不是一件好事了。要学会用上面的办法改正这种缺点。
    • 我决定疏远这群朋友，他们确实很好，但是他们不能带给我更多的成长。

奋斗

  • 健康的重要性：健康对于人来说太过重要了，我的胃，我的眼睛，我的耳朵。我的腿。我的一切的一切都要持续陪伴我走过很多地方，走过艰难的岁月和荣耀的岁月。
  • 对于英语的态度：对于我的人生发展来说，我分配给英语的时间只能多，不能少。只能更多而不能更少。英语是要持之以恒学习的科目，要伴随我一生的语言。

梦想

  • 梦想中的假期：春光灿烂的时候用一个月的时间，去一个美丽的，有一扇大落地窗的地方，读书学习，睡懒觉，写东西。
  • 梦想中的房子：梦想中的房子一定要高，要有一个大大的书房，有三面墙的书，每一本书我都看过，有着详细的笔记，有一张大大的灰色或者暖色的简约书桌，有一个抽屉，放着必要的学习用具，有一盏台灯，简约漂亮，背对一整块的大落地窗，可以看到壮阔的风景。不是书柜，只是钉在墙上的一排一排大理石乳白色的横木条间隔，与我的人一般高，间隔是A3纸那么大不同蹲下来就可以拿到底下的书，那个房子不一定要是我的，只要是一段时间属于我便好，那些书柜都可以拆卸带走
  • 对于自己最重要的话：我才21岁，才开始我的人生，即使我以前不够努力，也不够聪明，但是我知道现在的我不同以往。我还要机会，我还可以通过努力实现我的所有梦想，不要让阶级固化成了你不努力的不成功的借口。想去哪？想成为什么样的人，都要大胆的说出来，要努力的去追求。
  • 能够明显感觉到自己的期望和实际差距很大，造成了一定程度的焦虑。
    梦想与现实总是有一定的距离，对于目前的我来说是，巨大的距离，所以更要学会控制自己的情绪，按部就班，脚踏实地一步步来。最终，岁月会报答你的努力。
  • 梦想中一定要完成的事情：考研结束以后一定要拿出一段完整的时间，将以前的日记全部导入，然后写回忆录，然后存成电子档。
  • 优秀的人值得被记住：https://www.forbes.com/midas/list/ 福布斯最佳投资人榜 我敬重这些人

杂记

  • 哈利波特：感觉上大学，在成为一个研究生，就像是在霍格沃兹的过程，一步步成长，一步步走向成熟。我们都是那本书里面的一个NPC，我们是如此的普通，而我们又是如此的独特。实验就像是实战，而其他的课程就像是理论课程。
    • 伏地魔就像是隐喻希特勒，一个纯血种论，可是自己就是一个混血，真是讽刺。还有人写和伏地魔的同人小说，就因为那个人强大，或者帅气，就可以抛却自己的三观，无视那些鲜血吗？即使是从小的时候开始养成又如何，一个真正追求强大的人，骨子里就是淡漠的，不会因为任何人阻挡自己的路，即使是自己最亲的人，所以他们只会报答那些恩情，然后继续走自己决定的路，爱情或者亲情只是这段漫长人生的调剂。我讨厌自己的恋爱脑。
    • 所以结束吧，结束所有的同人小说。
  • 生活就像是一颗石子，投进了我平淡无痕的冥想池。
  • 上个星期天没有出去学习，这个星期天没有出去学习，我是要造反呀！
  • 如果我长时间没有思考，等待我的就是固步自封的灵魂死亡
  • 小说不同于工具书，是可以从情感上塑造一个人的，而工具书是从理智上塑造一个人。
  • 自己的博客不能泄露，里面有关于自己的私人信息，以后等到自己想要开放的时候，就把一些博客文章锁住。
  • 但凡是网上的信息，全以 NX 身份显示，包括自己意识不到的一些信息都要如此。
  • 混乱的思想一旦付诸笔下，便会理清
  • 不要对这个世界失去希望，你认为的世界有可能只是那座井。你是蹲坐在其中的青蛙。
    这个世界总有一些优秀的有素质的人存在 打电话的时候将不是自己教室的门关上
    有的人真的很吵，如果不能适应，那就离开。离开之前是否要给她一个教训，或者说看着他跌入深渊，当然是看着他跌入深渊。
    教室自习总是会有一些特别吵的人
  • 日出日落皆是一样的寂寥，其实不通过时间分辨，不通过心情分辨，便是一样的景色，就像是新生和垂垂老矣。
  • 出远门求学似乎不再是一件难事，但我们在心里上似乎将其看的太过轻松了。
  • 商家的恶性竞争在消费者看来是一种有利的事情。锅盔的故事让我想起了麦当劳和肯德基。站在商家的角度要如何处理这件事情。站在消费者的角度又要如何考虑
  • 傅雷的话，盛名之下其实难副，这个虽然我还没有经历，但是偶尔的成功也会让我处于一种这样的状态，那到底应该在出名的情况下处理这样的事情，或者要如何合适的面对，最近看到的哈利波特中波特成名以后的生活也让人遐想
  • 不患寡而患不均，张宇老师讲的概念，很有意思。这是一个很好的管理理念。来源于何处。要如何多一点的了解
  • 自助精神会不会对于商家促销有好的影响，自助冰淇淋
  • 如果有问题提出来，处理办法应该是：找目前该研究领域的知识来解答，如果找不到，那就自己解决
  • 爱情在大学期间真的是理智的吗？一起学习，参加考研，会减少考研期间的痛苦和无聊吗?
  • 如果想要了解本国真正的情况，应该去看对于本国无偏见的其他国家报道，这里的偏见包含好的偏见和坏的偏见，那些无意识的报道才是最真实的。
  • 过去的一些事情总是在攻击自己，让我无可适从。写下来，以后一起记录那些难过的事情，万邦财经，电航，pwc，还有买衣服砍价那些事情。20190404 20190423现在看来就没有一点感觉。
  • 最恐惧的不是那件事情，而是恐惧这种情绪。最思念的不是那个人，而是思念那种苦涩的味道。
  • 每一次成长都伴随着巨大的，决绝的代价。有些人说不想长大，不是指年龄的增长，而是不想经历那种成长的剧痛。
