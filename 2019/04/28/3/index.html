优雅夏奈尔，经典纪梵希 风格搭配 BR0 1.19

  • 身体 模板
    • 因此在穿衣打扮的时候，首先要考虑的不该是服装本身，而是想要达到怎样的效果。具体来说，服装应该以下面的顺序来进行挑选。在衬托娇小脸型、修长腿部的同时，还要考虑个人评委的突出或是既能拓展肩部，又能修长美腿，像这样整体规划好自己的想法再挑选衣服的话，就能穿出自己的个性风格。
    • 西方的时尚设计师们辛辛苦苦找到了解决方法-错失法。他或将裤子、裙子的腰线与边缘线巧妙的加以调整，或是将外衣的腰线上提以改善腿部的视觉效果，又或是将领口扩大让颈部看上去修长，以弥补身材上的不足。
    • 虽然都是服装，但东方和西方所注重的方向截然不同。包括韩国在内的东南亚各国，忠实从服装本身出发；而西方国家则重视从身体出发。但令人心痛的是，现在我们完全抄袭了西方的穿法。称其心痛是因为，东方人很难消化这种拥有8头身比例的西方穿着打扮方法。东方人比西方人的腰部长且腿部短。
    • 长腿是诠释身体之美的金钥匙。因此穿衣是有限考虑的应该是下肢的长度。最大限度地将腿部拉长，再考虑上身的装扮就容易多了。也就是说要拉伸人体比例的核心部分。
      • 下肢的长度是指腰部到腿底间的距离，由此来估量判断穿衣人的身材如何。
      • 将裤子的腰线稍稍上提，或是将它的脚线微微下拽，都能让腿部看上去更加细长。阔腿裤是通过调整脚线的位置，使腿部看上去更加细长。而高腰裤则是通过提升腰线来达到这个目的。连衣裙也是如此。连衣裙腰线的错视效果开始起作用。稍微的提高腰线，那么腰线以下的部分自然就变长许多。只是注意裙子的长度不能过短，因为会暴露臀部的位置。以上是通过调整腰线和脚线的位置来改变视觉效果的。
      • 反过来，用完全模糊化的方式处理它们，也会产生腿部的错视效果。高腰型连衣裙和袋型裤
    • 时尚潮流的全部并不仅是强调女性身体的妩媚。性感美固然重要，但贤淑秀丽的知性美也是散发女人味的重要因素之一。如果前者看作是肉体之美的话，那么后者就是精神之美。间或也需要体现女性的精神魅力，那么这时用一副遮盖这些灵动的曲线就似乎稳妥得多。
    • 上半身：颈部竖直向下的流向 肩部 水平流向 手臂 自然下垂 胸部 锁骨 性感。衬衣或者夹克遵循上身构造
    • 细腰
    • 骨盆：性感之美。由肌肉形成的臀部有很强的弹力性和塑造性，就像往碗里倒水一样，只要选好裤子的形状，即使再难看的臀部，也能让其散发性感之美。通过裤子来表现臀部形态的不同，或是缩小或是增翘。
    • 腿：与骨盆相连的大腿骨的角度稍稍扩开的话，就会加大两膝盖之间的距离，也就容易形成常说的O型腿。像这种情况，穿裤管宽或是长的裤子比较好。包围大腿骨的最外层肌肉与腿部肌肉相连，会让下身看上去更宽大。女性的情况是两部分容易混为一体，因此下身比男性略宽大。像这样的情况，只要选择臀部稍宽松，并且最外扩的位置高于大腿骨高度的裤子或裙子，就会有缩小臀部的功效。
    • 脚：三维曲面，类似于高跟鞋
  • 衣服 容器
    • 衣服的首要作用是保护身体，次要才是展现美丽。
    • 轮廓决定风格
      • X： 克里斯汀·迪奥 肩部和臀部稍微夸大，腰部紧收 性感
      • A：下部比肩膀宽大 优雅和可爱之间
      • H：夏奈尔 从肩部开始，用直线垂下的服装包裹整个身体 知性
      • O：无限自由 可爱 选择O型外廓的服装时，最好搭配紧腿裤或是紧身针织衫，也可以在O型外廓大衣下面搭配一条底边宽大的A型长裙
    • 学会打扮最有效方法是先了解贯穿于造型艺术中普遍存在的原理，然后再具体的分析各种类型服装。而与服装相关的造型原理有比例、对比、对称以及非对称等。
      • 比例：调整身体比例；调整服装比例
      • 对比：将属性完全相反的东西放到一起，从而把视觉效果最大化。面积对比、长度对比、色彩对比、华丽与单纯、不同材质对比（质感对比）
      • 对称：在任何情况下将物体形态处理平静安宁，造型上牢固的稳定性。给人权威和硬朗感
      • 非对称：胸针围巾 对称+非对称 非对称+非对称
  • 上衣的挑选
    • 领口：相框，三角形和圆形 选择上衣最该重视的就是上衣的领口 领口形态：暴露颈部和掩盖颈部 领口剪裁的流动趋势：横向和竖向
    • 肩部轮廓：对衣服的外廓产生巨大的影响；柔和的曲线形态，夸张的蓬起（权力套装，力量强壮）泡泡袖可爱。
    • 腰线：
      • 收腰类：x型外廓 腰线的位置间接暗示了腿的长度
      • 竖直类：运动灵活，典雅知性。没有收腰的特殊效果，过大的尺寸将身体的特征全部遮住，这就给观者带来无尽的遐想
    • 上衣的穿着
      • 夹克：分类：源自于男装的结实平和的构造；夸大女性的曲线，柔和性感；剩下的介于两者之间。体型分为胖和瘦两种。体型与夹克反向搭配。修饰形体；凸显个性；连接上下身的曲线流动
        • 翻领：门面 大V（脸部稍大的人，中间空白可以搭配自己的内衬）和小v（视线集中在脸部，头大或者脸圆的人要远离）；翻领的形态：直线和曲线 翻领的宽窄
        • 肩部：决定上端形象轮廓
        • 上下搭配：将夹克与下装进行协调搭配的时候，最重要的问题是要显得下身更长；腿的长度从腰线开始。夹克后部遮挡臀部
          • 长度结束于胸部的短款夹克：裸胸的连衣裙；收腰的连衣裙-X型
          • 长度结束于腰部的短款夹克：最难形成错视效果裙子底摆大幅加宽的A型裙子
          • 长度遮挡臀部的夹克
          • 夹克与休闲装的搭配
        • 衬衣：t字型主体 领子 袖子（袖管和袖口）
          • 贴身剪裁
          • 松弛舒适
          • 有一件白色衬衣和黑色衬衣
        • T恤：宽松舒适
          • 腰部系腰带 s型魅力
        • 马甲：端庄有格调
          • 搭配牛仔裤 提升品位
          • 毛衣马甲
          • 针织马甲
          • 里外搭配：简单T恤和牛仔裤+马甲 对比效果
  • 下衣的挑选
    • 腰线：高腰就是通过提高腰线让腿部更加修长的设计理念
    • 长度：关乎形象。盖住脚踝（端庄），结束于大腿（性感），膝盖是分界点。穿裤子：两部分；穿裙子：两部分或者三部分。
    • 宽度：塑造个性，宽松和紧身
  • 下衣的穿着
    • 裤子：打底裤（自己的腿要好看），直筒型（干净利落），喇叭裤，阔腿裤（修饰线条），窄腿裤（让腿部更长），灯笼裤（让腿部看上去不胖）。
    • 裙子：收紧型和宽大型
      • 郁金香型-性感 直筒型-端庄
      • a字群 喇叭裙 钟型群
  • 连衣裙 大衣 轮廓是关键

    • 项链围巾腰带搭配
